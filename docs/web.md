## App: community-web

### Summary
Vue3 web application for managing community markers on a map. Useful for location-based community events where several locations are involved.

### Current Scope and Limitations
- Limited address lists, such as a single town.

### Dependencies
- Requires back-end service to managing and maintaining markers.
